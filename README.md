# Ys IV: Mask of the Sun - A New Theory (PS2) English Translation [SLPM-65926]
English Translation of Ys IV: Mask of the Sun - A New Theory on PS2

# Notes
- Game uses PAK/PKI/PKM archive formatting as *Guilty Gear 2: Overdrive* (both games were primarily developed by Arc System Works)
- Graphics use TIM2 files
- Game uses Shift-JIS with ASCII support
